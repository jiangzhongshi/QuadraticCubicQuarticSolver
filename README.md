# QuadraticCubicQuarticSolver
C++ porting of ACM TOMS [Algorithm 954: An Accurate and Efficient Cubic and Quartic Equation Solver for Physical Applications](https://doi.org/10.1145/2699468)

Original source code by the authors is in Fortran, in the folder [954](954). 
It is under ACM Software License Agreement: http://toms.acm.org/algorithms-policy.cfm

We try to rewrite it in C++ or be further integrated in our library.
